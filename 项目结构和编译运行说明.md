# BlackholeWidget - 项目结构和编译运行说明

> Tauri v2 + Vue 3 桌面小组件：一个悬浮在桌面上的 3D 旋转黑洞，将文件拖到黑洞上即可删除。

---

## 1. 项目结构

```
BlackholeWidget/
├── index.html                       # 主窗口入口 HTML
├── settings.html                    # 设置窗口入口 HTML
├── package.json                     # Node.js 依赖和脚本
├── vite.config.js                   # Vite 多页面构建配置
├── generate-icons.py                # 图标生成脚本（生成 PNG / ICO）
│
├── src/                             # === Vue 前端源码 ===
│   ├── main.js                      # 主窗口 Vue 应用入口
│   ├── App.vue                      # 主窗口根组件（Three.js 画布 + 拖放叠加层）
│   ├── settings-main.js             # 设置窗口 Vue 应用入口
│   ├── SettingsApp.vue              # 设置窗口组件（确认删除 / 永久删除开关）
│   │
│   ├── composables/                 # 可组合逻辑
│   │   ├── useBlackHole.js          # Three.js 场景初始化 + 动画循环
│   │   └── useFileDrop.js           # Tauri 文件拖放事件处理 + 删除流程
│   │
│   ├── shaders/                     # GLSL 着色器

│   │
│   └── styles/

│

    ├── Cargo.toml                   # Rust 依赖配置
    ├── build.rs                     # Tauri 构建脚本
    ├── tauri.conf.json              # Tauri 窗口配置（无边框、透明、置顶、托盘）
    │
    ├── capabilities/
    │   └── default.json             # 权限声明（窗口、对话框、自动启动、事件等）
    │
    ├── icons/                       # 应用图标
    │   ├── 32x32.png
    │   ├── 128x128.png
    │   ├── icon.png
    │   ├── icon.ico                 # Windows 应用图标
    │   └── tray-icon.png            # 系统托盘图标
    │
    └── src/                         # Rust 源码
        ├── main.rs                  # 应用入口：注册命令、插件、托盘
        ├── config.rs                # 设置项 JSON 读写（%APPDATA%/blackhole-widget/）
        ├── file_ops.rs              # 文件删除：回收站（trash）或永久删除（std::fs）
        └── tray.rs                  # 系统托盘：图标（内存生成）、菜单、事件处理
```

## 2. 技术栈

| 层 | 技术 | 用途 |
|---|---|---|
| 桌面框架 | Tauri v2 | 窗口管理、系统托盘、原生 API、打包 |
| 3D 渲染 | Three.js | 黑洞 3D 场景渲染 |
| UI | Vue 3 (Composition API) | 组件逻辑 |
| 构建 | Vite 6 | 前端构建和热更新 |
| 后端 | Rust (Edition 2021) | 文件操作、设置持久化、自动启动 |
| 文件删除 | `trash` crate | 跨平台回收站操作 |
| 自动启动 | `tauri-plugin-autostart` | Windows 注册表开机启动 |
| 原生对话框 | `tauri-plugin-dialog` | 删除确认弹窗 |
| 着色器 | GLSL | Fresnel 边缘辉光效果 |

## 5. 功能说明

### 主窗口
- 无边框、透明、始终置顶的 320x320 浮动窗口
- 显示一个旋转的 3D 黑洞（由 Three.js 渲染）
- 左键拖拽窗口任意位置可移动
- 不显示在任务栏，仅通过系统托盘管理

### 文件拖放删除
- 从文件资源管理器拖放文件到黑洞上
- 拖入时：黑洞辉光增强 + 显示 "释放文件以删除" 提示
- 释放时：
  - 如开启 "删除前提示确认" → 弹出原生确认对话框
  - 如开启 "永久删除" → 直接删除（不入回收站）
  - 否则 → 移入回收站
- 删除后黑洞触发脉冲动画反馈

### 系统托盘菜单
```
[黑洞图标]     ← 左键点击：切换显示/隐藏
├── 显示/隐藏
├── 开机启动 ✓
├── 设置
└── 退出
```

### 设置窗口
- 通过托盘菜单 "设置" 打开
- 两个开关项：
  - **删除前提示确认**：拖放文件后弹出确认对话框
  - **永久删除**：不经回收站，直接从磁盘删除
- 设置立即保存到 `%APPDATA%\blackhole-widget\settings.json`

### 开机启动
- 通过 `tauri-plugin-autostart` 注册到 Windows 注册表
- 路径：`HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`
- 托盘菜单中 "开机启动" 可勾选/取消

## 6. 3D 黑洞渲染说明

黑洞由四个图层叠加渲染（Three.js + 自定义 GLSL 着色器），背景完全透明：

| 图层 | 几何体 | 材质 | 效果 |
|---|---|---|---|
| 事件视界 | `SphereGeometry` | `ShaderMaterial`（Fresnel） | 中心纯黑 → 边缘紫色辉光 |
| 吸积盘 | `TorusGeometry` x2 | `MeshBasicMaterial` + AdditiveBlending | 橙红发光环，倾斜 63°，缓慢旋转 |
| 光子环 | `TorusGeometry`（细） | `MeshBasicMaterial` + AdditiveBlending | 紧贴黑洞的薄亮环，呼吸脉冲 |
| 粒子云 | `Points`（400点） | `PointsMaterial` + AdditiveBlending | 盘面散布的橙红粒子，缓慢旋转 |

关键着色器逻辑（`blackHole.frag.glsl`）：
```glsl
// Fresnel 边缘效果：rim = 0 at 中心, 1 at 边缘
float rim = 1.0 - abs(dot(vNormal, vViewDir));
rim = pow(rim, uRimPower);        // 默认 3.5
float glow = rim * uGlowIntensity;

// 中心纯黑 → 边缘紫色
vec3 color = mix(vec3(0.0), uGlowColor, glow);

// 透明度：中心不透明 → 边缘半透明
float alpha = mix(0.95, 0.15, rim);
```

透明渲染器配置：
```js
renderer = new THREE.WebGLRenderer({
    alpha: true,
    premultipliedAlpha: false,    // 避免半透明区域偏白
    antialias: true,
});
renderer.setClearColor(0x000000, 0);  // 完全透明背景
```

## 7. 数据流

```
用户拖放文件
    ↓
Tauri onDragDropEvent
    ↓
useFileDrop.js → 读取设置 → (可选) 确认对话框
    ↓
invoke('delete_files', { paths })
    ↓
Rust main.rs → 读取 settings.json → file_ops.rs
    ↓
trash::delete() 或 std::fs::remove_file()
    ↓
返回结果 → 前端脉冲动画
```

## 8. 设置文件格式

存储路径：`%APPDATA%\blackhole-widget\settings.json`

```json
{
  "confirm_delete": true,
  "permanent_delete": false
}
```

首次运行自动创建，默认值均为安全选项（确认开启、回收站删除）。

## 9. 常见问题

**Q: 开发模式启动失败，提示缺少 WebView2？**
A: Windows 10/11 已内置 WebView2。如果是精简版系统，需从 [Microsoft](https://developer.microsoft.com/microsoft-edge/webview2/) 安装。

**Q: 窗口不透明 / 背景不是透明的？**
A: 确认 `tauri.conf.json` 中 `transparent: true`，且系统和显卡驱动支持窗口透明。

**Q: 拖放文件没有反应？**
A: 确保窗口可见（未隐藏），且 `dragDropEnabled: true` 在配置中已设置。部分安全软件可能拦截拖放操作。

**Q: 如何更换图标？**
A: 替换 `src-tauri/icons/` 下的 PNG/ICO 文件，或运行 `python generate-icons.py` 重新生成。

**Q: 如何彻底卸载？**
A: 卸载程序后用注册表编辑器删除 `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run` 中的 `blackhole-widget` 键值。
